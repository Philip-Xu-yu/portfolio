## 部署到GitHub Pages

1. 创建GitHub仓库 `portfolio`
2. 推送代码到仓库
3. 在仓库设置中启用GitHub Pages
4. 选择main分支作为源
5. 访问 https://yourusername.github.io/portfolio/
