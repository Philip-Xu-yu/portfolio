# 上传一份文档，AI就能回答所有问题 — 知识库问答助手开发实录

> 案例拆解：AI知识库问答助手
> 发布日期：2026-06-12

---

## 背景

一个做电商的朋友问我："我有500个FAQ，客户问的时候我一个个复制粘贴，能不能让AI自动回答？"

能。我花了一天做了一个知识库问答助手：上传文档，AI自动学习，然后基于文档内容回答问题。

---

## 解决的问题

| 问题         | 之前         | 之后       |
| ------------ | ------------ | ---------- |
| 回答客户问题 | 人工复制粘贴 | AI自动回答 |
| 新员工培训   | 读100页手册  | 直接问AI   |
| 内部知识查找 | 翻文档找半天 | 秒级响应   |

---

## 核心功能

**1. 文档上传**

- 支持PDF、TXT、Markdown
- 自动解析文档内容
- 自动分片建立索引

**2. 智能问答**

- 基于文档内容回答
- 支持多轮对话
- 显示答案来源（引用原文）

**3. 边界控制**

- 超出文档范围的问题，诚实说"不知道"
- 不会编造信息
- 引导用户联系人工客服

---

## 技术实现

```
后端：FastAPI
AI：OpenAI GPT-4o-mini
向量数据库：FAISS（轻量级）
文档解析：PyPDF2 + 自定义解析器
前端：原生HTML
```

核心流程：

```
上传文档 → 文本提取 → 分片 → 向量化 → 存储
用户提问 → 向量检索 → 找到相关片段 → AI生成回答
```
