# 餐厅老板必看：用AI自动回复大众点评差评

> 发布日期：2026-06-12
> 系列：行业AI工具

---

## 餐厅老板的痛点

- 大众点评有差评，不知道怎么回复
- 回复太生硬，客户更生气
- 回复太卑微，显得没底气
- 每天花30分钟想回复话术

## AI解决方案

**1. AI差评回复助手**

- 输入差评内容
- AI生成3种回复方案
- 专业、有温度、不卑微

**2. AI好评回复助手**

- 输入好评内容
- AI生成感谢话术
- 引导客户再次光临

**3. AI菜品描述生成器**

- 输入菜品信息
- AI生成诱人的菜品描述
- 适合菜单、外卖平台、小红书

**4. AI营销文案生成器**

- 输入活动信息
- AI生成朋友圈、小红书文案
- 吸引新客户

## 效果

| 任务       | 之前      | 之后     |
| ---------- | --------- | -------- |
| 回复差评   | 30分钟/条 | 2分钟/条 |
| 回复好评   | 5分钟/条  | 30秒/条  |
| 写菜品描述 | 15分钟/道 | 2分钟/道 |
| 写营销文案 | 30分钟    | 5分钟    |

## 总结

餐厅的核心是菜品和服务，不是写文案。AI帮你省下写东西的时间，让你专注于做好菜。
