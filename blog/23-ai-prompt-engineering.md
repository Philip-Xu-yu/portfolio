# 提示词工程入门：让AI输出更好结果的5个技巧

> 发布日期：2026-06-12
> 系列：AI技巧

---

## 为什么需要提示词工程？

同样的AI，不同的提示词，输出结果天差地别。

好的提示词：AI输出你想要的结果
差的提示词：AI输出一堆废话

## 5个技巧

**1. 明确角色**

```
差：帮我写一篇文案
好：你是一个小红书爆款文案写手，请帮我写一篇关于AI工具的文案
```

**2. 给出示例**

```
差：帮我写标题
好：请参考以下标题风格，帮我写5个标题：
- 《打工人必看！这5个AI工具让我效率翻倍》
- 《我用AI一天干完了3天的活》
```

**3. 指定格式**

```
差：分析一下这个市场
好：请用以下格式分析市场：
1. 市场规模
2. 增长率
3. 主要玩家
4. 机会点
5. 风险
```

**4. 限制范围**

```
差：帮我写一篇文章
好：帮我写一篇800字的小红书文章，主题是AI工具推荐，风格口语化，包含emoji
```

**5. 迭代优化**

```
第一次输出不满意？不要重新写提示词，而是在原来的基础上修改：
"很好，但请把标题改得更吸引眼球"
"请把第三点展开，加入具体案例"
```

## 总结

提示词工程不是玄学，是沟通技巧。你把需求说得越清楚，AI输出的结果越好。
