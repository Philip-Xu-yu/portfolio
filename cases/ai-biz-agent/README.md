# AI商业机会分析Agent

> 输入行业和条件，AI自动输出市场分析、竞品对比、商业模式画布和行动建议

---

## 功能

- 市场机会分析：市场规模、增长率、机会点评分
- 竞品分析：功能对比、优劣势、差异化机会
- 商业模式画布：自动生成标准9格画布
- 行动建议：30天计划、预算分配、里程碑

## 分析维度

| 维度       | 输出内容                                  |
| ---------- | ----------------------------------------- |
| 市场概况   | 规模、增长率、驱动因素                    |
| 用户画像   | 痛点、需求、行为特征                      |
| 机会点评分 | 5维度评分（市场规模/竞争/门槛/盈利/推荐） |
| 竞品对比   | 功能/用户/营收/优劣势                     |
| 商业画布   | 9格标准商业模式画布                       |
| 行动计划   | 30天分阶段执行方案                        |

## 技术栈

- 后端：FastAPI
- AI：OpenAI GPT-4o-mini
- 分析框架：商业模式画布 + SWOT
- 前端：HTML + CSS

## 快速启动

```bash
pip install -r requirements.txt
export OPENAI_API_KEY="your-key"
python app.py
```

访问 http://localhost:8000
