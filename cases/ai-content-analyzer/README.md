# AI爆款内容分析器

> 输入内容 → 分析爆款因素 → 生成优化建议 → 仿写生成

## 功能

- **内容分析**：标题吸引力、内容结构、情绪触发点、平台适配度、爆款评分
- **优化建议**：针对不足给出具体优化方案，含优化前后对比
- **仿写生成**：保留核心信息，用爆款风格重新撰写

## 适用人群

- 内容创作者（想提升内容质量）
- 自媒体运营（想分析竞品爆款）
- 营销人员（想优化文案效果）

## 技术栈

- **后端**：FastAPI + OpenAI API
- **前端**：内嵌HTML（单文件部署）
- **部署**：Render / Railway / 本地

## 快速开始

```bash
pip install -r requirements.txt
export OPENAI_API_KEY="your-key-here"
python app.py
```

访问 `http://localhost:8001` 即可使用。

## API接口

| 接口            | 方法 | 说明             |
| --------------- | ---- | ---------------- |
| `/api/analyze`  | POST | 分析内容爆款因素 |
| `/api/optimize` | POST | 生成优化建议     |
| `/api/rewrite`  | POST | 仿写生成         |
