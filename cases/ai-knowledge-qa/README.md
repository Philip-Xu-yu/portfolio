# AI知识库问答助手

> 上传文档 → 自动建立知识库 → 基于文档内容回答问题

## 功能

- **文档上传**：支持粘贴任意文本内容（产品手册、FAQ、论文等）
- **自动分片**：将长文档自动分割成小块，建立可检索的知识库
- **智能问答**：基于文档内容回答问题，支持引用原文
- **多知识库**：支持同时管理多个知识库

## 适用人群

- 企业客服（基于产品手册自动回答客户问题）
- 研究人员（基于论文快速检索信息）
- 教育培训（基于教材回答学生问题）
- 任何需要从大量文档中快速获取信息的人

## 技术栈

- **后端**：FastAPI + OpenAI API
- **前端**：内嵌HTML（单文件部署）
- **部署**：Render / Railway / 本地

## 快速开始

```bash
pip install -r requirements.txt
export OPENAI_API_KEY="your-key-here"
python app.py
```

访问 `http://localhost:8002` 即可使用。

## API接口

| 接口          | 方法 | 说明               |
| ------------- | ---- | ------------------ |
| `/api/upload` | POST | 上传文档建立知识库 |
| `/api/ask`    | POST | 基于知识库提问     |
