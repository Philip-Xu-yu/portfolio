# AI自媒体起号助手

> 输入领域/赛道 → 10分钟生成30天内容规划

## 功能

- **账号定位**：输入领域，自动生成账号人设、内容方向、差异化定位、变现路径
- **爆款选题**：输入热点或随机生成10个有爆款潜力的选题
- **完整文案**：选题 → 完整文案（标题、正文、标签），支持多种风格

## 适用人群

- 想做自媒体但不知道从哪开始的人
- 想批量生产内容的自媒体运营
- 想快速起号的新手

## 技术栈

- **后端**：FastAPI + OpenAI API
- **前端**：内嵌HTML（单文件部署）
- **部署**：Render / Railway / 本地

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 设置环境变量
export OPENAI_API_KEY="your-key-here"

# 启动
python app.py
```

访问 `http://localhost:8000` 即可使用。

## API接口

| 接口                | 方法 | 说明             |
| ------------------- | ---- | ---------------- |
| `/api/account-plan` | POST | 生成账号定位方案 |
| `/api/topics`       | POST | 生成爆款选题     |
| `/api/article`      | POST | 生成完整文案     |
