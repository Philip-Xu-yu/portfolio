# 深度案例2：AI内容生产平台

> 批量生成多平台内容：选题、文案、标题、标签

---

## 功能

- 选题生成：输入领域，批量生成选题
- 文案生成：输入选题，生成多平台文案
- 标题优化：生成5个备选标题
- 标签匹配：自动匹配热门标签
- 批量处理：一次生成10篇内容
- 内容分析：分析文案爆款因素

## 支持平台

- 小红书
- 抖音
- 公众号
- B站
- 微博

## 技术栈

- 后端：FastAPI
- AI：OpenAI API
- 前端：HTML + CSS + JS

## 快速启动

```bash
pip install -r requirements.txt
export OPENAI_API_KEY="your-key"
python app.py
```
