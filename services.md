## SKU 1：基础版 — ¥99

### 服务内容

帮客户定制一个简单的AI工具，解决一个具体问题。

可选范围：

- AI聊天机器人（基于FAQ自动回复）
- AI自动化脚本（数据处理、内容生成）
- AI文案生成器（特定场景）

### 交付物

- 可运行的AI工具（部署到Render/ Railway）
- 可访问的URL
- 使用文档（README）
- 7天售后支持

### 交付周期

1-2天

### 目标客户

- 电商小卖家（需要FAQ自动回复）
- 自媒体人（需要内容生成工具）
- 个体户（有一个重复性任务想自动化）

### 成交话术

```
场景1：客户问"能做什么"
→ "我可以帮你做一个AI工具，解决一个具体问题。比如自动回复客户常见问题、自动生成文案、自动处理数据。99元，1-2天交付，包部署包售后。"

场景2：客户问"和ChatGPT有什么区别"
→ "ChatGPT是通用工具，你需要每次手动输入。我帮你做的是一个专属工具，已经把你的业务知识训练进去了，打开就能用，不用每次教它。"

场景3：客户嫌贵
→ "99元相当于一顿火锅钱。但这个工具可以帮你每天省1-2小时，一个月就是30-60小时。你的时间值多少钱？"

场景4：客户犹豫
→ "你可以先看看我的案例（发Demo链接）。如果觉得有用，我们再聊。不满意不收钱。"
```

## 通用成交策略

### 信任建立

1. 发案例Demo链接（让客户看到真实效果）
2. 发GitHub仓库链接（证明技术能力）
3. 提供免费咨询（30分钟需求沟通）
4. 不满意不收钱（降低客户风险）

### 报价时机

- 先了解需求，再报价
- 不要在第一条消息就报价
- 确认客户有真实需求后再报价

### 付款方式

- 闲鱼：担保交易，客户放心
- 微信/支付宝：老客户直接转账
- Upwork/Fiverr：国际客户

### 售后策略

- 基础版：7天支持
- 标准版：14天支持
- 高级版：30天支持
- 超出支持期：按小时收费（¥100/小时）
